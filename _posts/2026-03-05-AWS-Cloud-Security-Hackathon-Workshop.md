AWS Cloud Security: Hackathon Workshop

AWS User Groups along the Colorado Front Range are hosting a hands-on security hacking workshop series focused on real-world AWS attack scenarios and defensive techniques.

![AWS Cloud Security Hackathon Workshop Invite](/assets/img/LouisvilleHackathonInvite.png)

## Event Details

- **Date:** March 5, 2026, 3:00 PM - 5:00 PM
- **Location:** 550 South McCaslin Blvd Unit B, Louisville, CO 80027

## What to Expect

Join us for a hands-on workshop hosted by nClouds where you'll work through real-world AWS attack scenarios and learn defensive techniques. This is a great opportunity to sharpen your cloud security skills in a collaborative, interactive environment.

Come ready to hack, learn, and have fun!
